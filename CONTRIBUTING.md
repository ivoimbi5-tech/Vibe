# Contribuindo para o VibeWake

Obrigado por seu interesse em contribuir para o VibeWake! Aqui estão algumas diretrizes para ajudar você a começar.

## Como Contribuir

1. **Relate Bugs:** Se você encontrar um bug, abra uma Issue descrevendo o problema e como reproduzi-lo.
2. **Sugira Melhorias:** Se você tiver uma ideia para uma nova funcionalidade, abra uma Issue para discutirmos.
3. **Envie Pull Requests:**
   - Faça um fork do repositório.
   - Crie uma branch para sua funcionalidade ou correção (`git checkout -b feature/nova-funcionalidade`).
   - Faça o commit de suas alterações (`git commit -m 'Adiciona nova funcionalidade'`).
   - Faça o push para a branch (`git push origin feature/nova-funcionalidade`).
   - Abra um Pull Request.

## Estilo de Código

- Use 2 espaços para indentação.
- Escreva código limpo e bem documentado.
- Siga as convenções de TypeScript e React.

## Licença

Ao contribuir, você concorda que suas contribuições serão licenciadas sob a licença MIT do projeto.
