# Política de Segurança

## Versões Suportadas

Atualmente, apenas a versão mais recente é suportada.

| Versão | Suportada |
| ------- | --------- |
| 1.0.x   | ✅        |
| < 1.0   | ❌        |

## Relatando uma Vulnerabilidade

Se você descobrir uma vulnerabilidade de segurança, por favor, não abra uma Issue pública. Em vez disso, envie um e-mail para [seu-email@exemplo.com].

Nós responderemos o mais rápido possível e trabalharemos com você para resolver o problema.
