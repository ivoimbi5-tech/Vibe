# VibeWake - AI Music Alarm

VibeWake é um aplicativo de despertador sofisticado que te acorda com suas músicas favoritas e citações inspiradoras.

## ✨ Funcionalidades

- **Despertadores Personalizados:** Configure múltiplos alarmes com horários e etiquetas específicas.
- **Playlists de Despertar:** Crie coleções de músicas para tocar em sequência quando o alarme tocar.
- **Citações Inspiradoras:** Receba uma das 100 citações motivadoras integradas ao acordar (funciona offline).
- **Biblioteca de Música:** Escolha entre trilhas padrão ou faça upload de seus próprios arquivos de áudio locais.
- **Interface Moderna:** Design elegante com efeitos de vidro (glassmorphism) e animações suaves.
- **Responsivo:** Funciona perfeitamente em dispositivos móveis e desktop.

## 🚀 Tecnologias Utilizadas

- **React 19**
- **TypeScript**
- **Tailwind CSS**
- **Vite**
- **Lucide React (Ícones)**

## 🛠️ Como Executar

### Pré-requisitos

- Node.js instalado

### Instalação

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/vibewake-ai-alarm.git
   cd vibewake-ai-alarm
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

4. Abra o navegador em `http://localhost:3000`.

## 📦 Build

Para gerar a versão de produção:
```bash
npm run build
```
